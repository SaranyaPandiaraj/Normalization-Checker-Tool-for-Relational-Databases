About



        Our Normalization Automated tool helps in the process of automating the relational database schema normalization up to BCNF form. This tool mainly concentrates on addressing the legacy system issues where we will be unaware of the physical design behind the system. The tool can handle the below Normal Forms and also helps you to identify the functional dependencies for the Legacy systems.

  • Functional Dependencies for Legacy Systems

  • First Normal Form (1NF)

  • Second Normal Form (2NF)

  • Third Normal Form (3NF)

  • Boyce-Codd Normal Form (BCNF)
